## 🔍 Overview
This project focuses on analyzing e-commerce sales data to uncover key revenue trends, customer behavior patterns, and operational inefficiencies. The goal is to derive actionable insights that can help improve business performance and decision-making.

---

## 🛠 Tech Stack
- Python (Pandas, Matplotlib, Seaborn)
- SQL (SQLite / pandasql)
- Power BI

---

## 📂 Project Workflow
1. Data Cleaning & Preprocessing using Python  
2. Exploratory Data Analysis (EDA) to identify trends and patterns  
3. SQL-based analysis for aggregations and KPI calculations  
4. Interactive dashboard creation using Power BI

## 📈 Key Metrics
- Total Revenue  
- Total Orders  
- Total Customers  
- Average Order Value (AOV)  
- Cancellation Rate  

---

## 🔑 Key Insights
- High cancellation rate leading to revenue leakage  
- Revenue is heavily concentrated in top product categories  
- Majority of customers are one-time buyers  
- Payment method trends highlight customer preferences  

---

## 🚀 Business Recommendations
- Improve operational efficiency to reduce order cancellations  
- Focus on high-performing product categories  
- Introduce strategies to improve customer retention  
- Optimize checkout experience based on preferred payment methods
